Difference between local storage , session storage and cookies

Answer :

Cookies are primarily used for user authentication and tracking, while Local Storage and Session Storage are used to store user data and preferences. Local Storage is intended for data that needs to persist between sessions, while Session Storage is intended for data that only needs to be available for the current session.

Question 2: Difference between map, filter, for each and find

Answer :

Map is used to transform each element of an array into a new value, filter is used to select elements that meet a certain condition, forEach is used to perform a function on each element of an array, and find is used to find the first element in an array that satisfies a certain condition.

Question 3: How does JavaScript Event loop work

Answer :

The event loop is a mechanism in JavaScript that allows the browser to efficiently handle multiple concurrent tasks by using a call stack to keep track of function calls and an event queue to hold events that have occurred but have not yet been processed. The event loop checks the event queue for events that are ready to be processed and executes the corresponding function on the call stack.

Question 5: how the chooseQuiz function works

Answer :

The chooseQuiz function updates an array of answers based on the user's input for a given quiz question. If the user has already answered the question before, the function updates the existing answer object in the array. If not, it adds a new answer object to the array. The function then calls another function to display the answers on the screen.
